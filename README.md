## D3 Business API – Mintlify docs

This repository contains the Mintlify documentation configuration and content for the **D3 Business API**.

- Config: `mint.json`
- Content root: `docs/`
  - `introduction.mdx`, `quickstart.mdx`
  - `business-api/*` – core Business API concepts and endpoints
  - `sdks/node.mdx` – official Node.js client usage

To use these docs with Mintlify, follow the Mintlify quickstart and point it at this repository as your documentation source. See the Mintlify docs for details: [`https://www.mintlify.com/docs`](https://www.mintlify.com/docs).
