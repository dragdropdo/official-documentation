## Key Capabilities

- Upload any supported file type to regional MinIO storage.
- Create **operations** such as convert, compress, merge, zip, share, lock/unlock/reset password.
- Get operation **status** via polling or **webhooks**.
- Manage **API keys** and **webhooks** through the D3 dashboard.

All of these capabilities are exposed via the `/v1/biz` Business API surface. API keys and webhooks are managed through the dashboard at [dragdropdo.com/auth/signin](https://dragdropdo.com/auth/signin) in your Account section.
